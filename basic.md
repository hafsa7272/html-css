# 🌐 Web Basics (HTML, CSS, href, HTTPS)

## 📌 1. HTML (HyperText Markup Language)

### 🔹 Full Form:

**HTML = HyperText Markup Language**

### 🔹 Kya hai?

HTML web pages ki **structure (dhancha)** banata hai.

### 🔹 Basic Example:

```html id="html1"
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>Yeh ek paragraph hai</p>
</body>
</html>
```

### 🔹 Important Tags:

* `<h1>` → Heading
* `<p>` → Paragraph
* `<a>` → Link
* `<img>` → Image

---

## 🎨 2. CSS (Cascading Style Sheets)

### 🔹 Full Form:

**CSS = Cascading Style Sheets**

### 🔹 Kya hai?

CSS web page ko **design aur style** deta hai.

### 🔹 Example:

```css id="css1"
body {
    background-color: lightgray;
}

h1 {
    color: blue;
}
```

---

## 🔗 3. href (Hyperlink Reference)

### 🔹 Full Form:

**href = Hypertext Reference**

### 🔹 Kya hai?

`href` links banane ke liye use hota hai.

### 🔹 Example:

```html id="href1"
<a href="https://www.google.com">Google</a>
```

### 🔹 Types:

* Absolute URL → `https://google.com`
* Relative URL → `about.html`

---

## 🔒 4. HTTPS (HyperText Transfer Protocol Secure)

### 🔹 Full Form:

**HTTPS = HyperText Transfer Protocol Secure**

### 🔹 Kya hai?

HTTPS ek **secure internet protocol** hai jo data ko encrypt karta hai.

### 🔹 Features:

* 🔐 Data safe hota hai
* 🔐 Hackers se protection
* 🌐 Trusted websites use karti hain

### 🔹 Example:

```
https://example.com
```
